# ShadowCon is a Real-Estate web application made in React and Next JS. 

# This Application has features of displaying homes for :

- **_Renting_**
- **_Buying_**

## Search on basis of filters

- Purpose
- Rent Frequency
- Min Price
- Max Price
- Max Area
- Rooms
- Baths
- Property Type

## Search by location

- Search your property by location
